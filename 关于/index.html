介绍

ThoseYearsSong

收集 90、00 后青春的歌。

整站架设在 Github Pages 服务上，由 BobYang 维护。

为啥这站点这么丑？

在写这个站点的时候是打算用比较靠近 2005 年左右的设计，但是莫名奇妙就写成了 2000 年代的设计，还不能在 IE 上正常访问。比较肯定的是这个 CSS 是 2022 年写的...

你这个站点资料有错昂！

如果发现站点资料有错误请发信至我的电子信箱或提出 issue，我会尽速修改。

你音源哪来的版权？该不会是侵权吧...

本站点音源都是从哔哩哔哩或油管上外链，本站点不储存关于音源的档案，亦不提供下载，故应不构成侵权。

我可以参与维护么？

当然可以，欢迎 Fork 我们位于 Github 的存储库并提交 Pull Requests。
